# 本地 BGM 目录

**试听或视频中若没有背景音**：请把 MP3 下载到本目录即可，无需改代码。

## 国内可访问的免费 BGM 来源（推荐）

1. **Pixabay 中文**（免版税、可商用）  
   https://pixabay.com/zh/music/  
   搜索「背景音乐」「轻音乐」「钢琴」等，下载 MP3 后按下面表格重命名。

2. **爱给网**（部分免费，需注册）  
   https://www.aigei.com/music/

3. **淘声网 / 飞声**（无版权音乐库）  
   搜索「免费可商用 BGM」即可找到多个站点。

## 使用步骤

1. 从上述任一站点下载你喜欢的曲目（MP3）。
2. 将 MP3 重命名为下表中对应文件名（如 `classical.mp3`），放入本目录 `public/bgm/`。
3. 刷新页面后，试听与生成视频都会使用本地文件。
